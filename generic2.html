About Me - Christian Aspinwall

Hi, I’m Christian Aspinwall — IT pro by day, martial artist by night 🥋

If you’re here, you probably care about networks and firewalls… but first, let me confess: I once accidentally rebooted a production server while testing a PowerShell script. (Don’t worry—I stayed up until 3 AM fixing it, fueled by cold pizza and Breaking Benjamin.)

🤖 Tech Journey: From Game Mods to Global Networks

My tech obsession started at 12 when I modded Skyrim so aggressively it crashed our family PC for a week. Today, I:

  • Argue with Cisco switches that occasionally argue back
  • Secretly judge companies that use default admin passwords (we all do it)
  • Still geek out over Linux terminal commands like it’s magic ✨

Certifications? Yes (CCNA, Security+, etc.). Real superpower? Fixing printers that “just stopped working” for no reason.
